# 🏠 Dari.ma

Projet de machine learning pour l’estimation des prix immobiliers au Maroc.

## 🚀 Fonctionnalités
- Scraping de données immobilières
- Nettoyage des données
- Modèle de prédiction (XGBoost)
- Interface Streamlit

## 🛠️ Technologies
- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit

## 📊 Objectif
Prédire le prix d’un bien immobilier en fonction de ses caractéristiques.
